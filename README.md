## Campus Cafe Management System

**The Campus Cafe Management System is a Python-based application designed to facilitate the management of a cafe within a 
campus environment. This system provides an efficient way to handle various cafe operations, including property management inside the cafe and and registration of new students to that cafe and allowing the student to access the cafe at bearkfast,
lunch and dinner once on each.**

## Features
- **Property Management**: Easily manage the cafe's property within the campus. This includes tracking details such as location, seating capacity, amenities, and operating hours.

- **Student Registration**: Register new students to the cafe system. Students can provide their personal information and be assigned a unique identification number for access purposes.

- **Meal Access Control**: Facilitate student access to the cafe for breakfast, lunch, and dinner. Once registered, students can use their identification number to enter the cafe during each respective mealtime.

## Installation 
1. Clone repository
 ```bash
 git clone https://github.com/NatnaelAbebe19/campus-cafe-management-system.git
```
2. Navigate to the project directory:
```bash
 cd campus-cafe-management-system
```
3. Install the required dependencies:
```bash
 pip install -r requirements.txt
```
4. Run the application:
```bash
 python main.py
```
## Usage
- Launch the application to access the main menu.

- From the menu, choose the desired options for property management, student registration, or meal access control.

- Follow the prompts and provide the necessary information as requested by the system.

- Once registered, students can use their assigned identification number to access the cafe during breakfast, lunch, and dinner hours.
